## [0.1.0] - 2026-03-22

### Added

- Initial public release of `@vanthdev/aiwind`.
- React primitives: Button, Card (compound), Label, Input, Textarea, Badge, Alert (compound), Separator, Skeleton, Switch, Spinner, Progress, `cn`.
- Storybook 8 with a11y addon; Tailwind v4-oriented styling (no bundled CSS).
- Documentation: README (consumers), CONTRIBUTING (maintainers), AGENTS (app/agent integration).
